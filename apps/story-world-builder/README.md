# Story World Builder

Build persistent fictional worlds with characters, locations, and lore.

## Features

- Tab-based UI: Characters, Locations, Lore, Chat
- Add entries with name, role/type, description, traits
- "Expand with AI" — enriches entries with context-aware detail
- "Ask About World" — chat tab for querying your world
- All data stored locally in SQLite
- AI maintains consistency across your world

## Permissions

- `ai` — AI-powered expansion and world chat
- `storage.kv` — App settings
- `storage.db` — SQLite for world data
