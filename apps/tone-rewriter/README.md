# Tone Rewriter

Rewrite any text in your chosen tone — formal, casual, bold, empathetic, humorous, or concise.

## Features

- Six tone options as selectable pills
- Real-time streaming rewrite
- Copy to clipboard
- Save and browse recent rewrites

## Permissions

- `ai` — AI-powered rewriting
- `storage.kv` — Save rewrites locally
- `clipboard` — Copy results
