# Changelog Writer

Turn git diffs and commit lists into clean, user-facing changelogs.

## Features

- Paste git diffs, commit logs, or change lists
- Three format options: Keep a Changelog, Conventional Commits, Bullet Points
- Optional version number for headers
- Copy to clipboard, save for later

## Permissions

- `ai` — AI-powered changelog generation
- `storage.kv` — Save changelogs locally
- `clipboard` — Copy results
