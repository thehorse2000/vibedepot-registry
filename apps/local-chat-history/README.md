# Local Chat History

A persistent AI chat that saves every conversation locally on your device.

## Features

- Multiple conversation threads
- Full chat history stored in local SQLite database
- Auto-titles conversations from first message
- Streaming AI responses
- Delete individual conversations
- All data stays on your device

## Permissions

- `ai` — AI chat
- `storage.kv` — App settings
- `storage.db` — SQLite for conversation storage
