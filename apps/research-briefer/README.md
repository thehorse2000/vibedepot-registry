# Research Briefer

AI-powered research briefing generator for VibeDepot.

## Features

- **Structured briefings** — Get organized reports with background, key findings, perspectives, and takeaways
- **Three depth levels** — Quick Brief, Standard, or Deep Dive depending on your needs
- **Focus areas** — Optionally steer the briefing toward specific aspects of a topic
- **Streaming output** — Watch the briefing generate in real-time with markdown formatting
- **Save briefings** — Store up to 15 briefings for later reference
- **Multi-provider** — Works with Anthropic, OpenAI, and Gemini

## Usage

1. Enter a research topic
2. Select a briefing depth (Quick Brief / Standard / Deep Dive)
3. Optionally add focus areas to narrow the scope
4. Click "Generate Briefing" and watch the structured report stream in
5. Copy or save the briefing for reference

## Permissions

- `ai` — Required for AI-powered briefing generation
- `storage.kv` — Used to persist saved briefings
- `network` — Allows network access for AI provider communication
