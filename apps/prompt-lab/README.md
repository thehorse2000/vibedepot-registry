# Prompt Lab

Test and iterate on AI prompts with variables. Save templates and compare runs.

## Features

- Write system prompts with {{variable}} placeholders
- Auto-detects variables and generates input fields
- Adjustable temperature and max tokens
- Save and load prompt templates
- Run history per template
- SQLite-backed storage

## Permissions

- `ai` — AI prompt execution
- `storage.kv` — App settings
- `storage.db` — SQLite for templates and run history
