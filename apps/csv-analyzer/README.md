# CSV Analyzer

Paste CSV data and ask questions in plain English. Get summaries, trends, and anomalies.

## Features

- Auto-preview with first 5 rows and column stats
- Quick action buttons: Summarize, Find Trends, Spot Anomalies
- Ask any question about your data
- Client-side CSV parsing

## Permissions

- `ai` — AI-powered data analysis
- `storage.kv` — Local storage
