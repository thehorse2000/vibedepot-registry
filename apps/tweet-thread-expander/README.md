# Tweet Thread Expander

Turn Twitter/X threads into polished long-form blog posts, newsletters, or articles.

## Features

- Paste tweet thread text
- Choose output style: blog post, newsletter, or article
- Streaming AI expansion with smooth transitions
- Copy to clipboard, save for later

## Permissions

- `ai` — AI-powered content expansion
- `storage.kv` — Save posts locally
- `network` — Network access
- `clipboard` — Copy results
