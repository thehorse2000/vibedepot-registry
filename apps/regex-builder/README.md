# Regex Builder

Describe what you want to match in plain English, get a working regex with test cases and explanation.

## Features

- Natural language to regex conversion
- Plain-English explanation of each regex part
- Auto-generated test cases (should match / should not match)
- Live regex tester — type strings to see instant match results
- Save and reuse patterns

## Permissions

- `ai` — AI-powered regex generation
- `storage.kv` — Save patterns locally
- `clipboard` — Copy regex to clipboard
