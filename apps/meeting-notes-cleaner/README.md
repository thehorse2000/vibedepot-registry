# Meeting Notes Cleaner

Clean up messy meeting notes into structured summaries with decisions, actions, and owners.

## Features

- Paste raw transcript or messy bullet notes
- AI extracts key decisions, action items with owners, and open questions
- Copy cleaned notes to clipboard
- Save and browse past cleanups

## Permissions

- `ai` — AI-powered note organization
- `storage.kv` — Save cleaned notes locally
- `clipboard` — Copy results to clipboard
