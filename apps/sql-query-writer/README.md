# SQL Query Writer

Describe what data you want in plain English, get a production-ready SQL query with explanation.

## Features

- Natural language to SQL conversion
- Optional schema input for accurate queries
- Supports PostgreSQL, MySQL, and SQLite
- Syntax-highlighted SQL output
- Copy SQL to clipboard
- Save and browse query history

## Permissions

- `ai` — AI-powered query generation
- `storage.kv` — Save queries locally
- `clipboard` — Copy SQL to clipboard
